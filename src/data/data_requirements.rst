
Required data sets
==================

This document lists the SOEP data files which are necessary to run this
project. We specifically refer to version 32.1 of the GSOEP and to both
components, GSOEPLong and GSOEPCore.


GSOEPLong
---------

- pl.dta
- pgen.dta


GSOEPCore
---------

- bioagel.dta
